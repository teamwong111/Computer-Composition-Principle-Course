## 简介
本仓库为包含2020年同济大学计算机组成原理课程的三个大作业：
1. 乘法器
2. 除法器
3. 54条指令CPU

│     ├─MULT.v  //带符号乘法器

│     └─mult_tb.v   //测试
├─DIV
│  ├─report
│  │  └─report.pdf
│  └─code
│     ├─DIV.v  //带符号除法器

├─CPU-54
│  ├─report
│  │  └─report.pdf
│  └─code
│     ├─alu.v
│     ├─controlunit.v
│     ├─cp0.v
│     ├─cpu.v
│     ├─dmem.v
│     ├─hi_lo.v
